The majority (200 or so) of these files are downloaded from the "Let's Move To..." articles in the property section of the Guardian 
(written mainly by Tom Dyckhoff (@tomdyckhoff on Twitter), though a few were written by Sophie Heawood (@heawood on Twitter)).

They are cut-and-pasted and lightly edited, but otherwise pretty much verbatim.

Originals available at:
https://www.theguardian.com/money/series/letsmoveto


The exceptions are:

- the file 'random_quotes.txt', which consists of quotes taken from the following sources in pycorpora:
	data\words\literature\technology_quotes.json
	data\words\oprah_quotes.json (Words of wisdom by Oprah Winfrey)
	data\words\proverbs.json (A list of proverbs sourced from http://tww.id.au/proverbs/proverbs.html)
	data\film-tv\Westworld_quotes.json (quotes from the TV show Westworld)
	data\words\us_president_quotes.json (quotes from US presidents)

- 'pqf2.txt', which is a heavily-edited version of the Pratchett Quote File v6.0.
	Original is available from https://www.lspace.org/books/pqf/index.html.

- 'da_quotes.txt', which is an edited list of quotes from Douglas Adams.
	Originals taken from:
	https://en.wikiquote.org/wiki/Douglas_Adams
	https://en.wikiquote.org/wiki/Last_Chance_to_See
	https://en.wikiquote.org/wiki/The_Hitchhiker%27s_Guide_to_the_Galaxy

- 'annals.txt', which is a lightly-edited version of the main body text of 
	'The Annals of the Parrigues' by Emily Short.
	Original avaliable as a PDF at:
	https://drive.google.com/file/d/0B97d5C256qbrOHFwSUhsZE4tU0k/view
	(read Emily Short's blog entry about it at:
	https://emshort.blog/tag/the-annals-of-the-parrigues/)

- 'revelation.txt'
	The Book of Revelation, from the Bible.
	Specifically, Modern English Version (MEV)
	The Holy Bible, Modern English Version
	Downloaded from https://www.bible.com/bible/1171/REV.1.MEV (onwards)
	(Edited solely to remove chapter and verse numbers.)
